VectorizedStatistics.vmaximum — Method
vmaximum(A; dims)

As Base.maximum, but vectorized: find the greatest value contained in A, optionally over dimensions specified by dims.

Examples

julia> using VectorizedStatistics

julia> A = [1 2; 3 4]
2×2 Matrix{Int64}:
 1  2
 3  4

julia> vmaximum(A, dims=1)
1×2 Matrix{Int64}:
 3  4

julia>  vmaximum(A, dims=2)
 2×1 Matrix{Int64}:
 2
 4

VectorizedStatistics.vmean — Method
vmean(A; dims)

As Statistics.mean, but vectorized: compute the mean of all elements in A, optionally over dimensions specified by dims.

Examples

 3  4

julia> vmean(A, dims=1)

julia> vmean(A, dims=2)
2×1 Matrix{Float64}:
